# Input Management

ViPPET supports three types of pipeline input sources, each managed through a dedicated section in the left-side menu:

- [Cameras](./input-management/cameras.md) — use connected USB or network cameras as input.
- [Image Sets](./input-management/images.md) — use uploaded image sets as input.
- [Videos](./input-management/videos.md) — use uploaded or built-in sample videos as input.
