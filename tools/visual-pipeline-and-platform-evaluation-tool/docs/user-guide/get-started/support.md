# Get Support

If you encounter a problem while using ViPPET, use the
[GitHub issue tracker](https://github.com/open-edge-platform/edge-ai-libraries/issues) to report it.

Before creating a new issue, review the existing issues to check whether the problem has already been reported.

When filing a new issue, add the `vippet` label and include enough detail to help reproduce and investigate
the problem, such as:

- ViPPET version
- Platform and operating system details
- Steps to reproduce the issue
- Expected behavior and actual behavior
- Relevant logs, screenshots, or error messages

Providing complete information helps the maintainers diagnose issues faster and respond more effectively.
